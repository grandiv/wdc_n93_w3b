## Getting Started

Pertama, unduh dan install node.js
https://nodejs.org/en

Kedua, setelah menginstall node.js, jalankan server pengembangan di terminal

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
# atau
bun dev
```

Buka [http://localhost:3000](http://localhost:3000) dengan browser anda (Chrome, Mozilla, Safari, dll).
